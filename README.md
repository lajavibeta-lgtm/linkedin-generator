# LinkedIn Content Generator

Generador de posts y artículos de LinkedIn con IA. Crea contenido profesional, auténtico y listo para publicar.

## Qué hace

- Genera posts cortos y artículos largos para LinkedIn
- Adapta el tono y estilo al perfil del autor
- Incluye opción de imagen generada automáticamente

## Cómo usar

Abre la app, ingresa un tema, selecciona el tipo de contenido y haz clic en generar.

## Deploy

Desplegado en Vercel. Variables de entorno requeridas en el dashboard de Vercel.
